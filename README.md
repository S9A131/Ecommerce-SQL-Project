# E-Commerce SQL Analytics Project

SQL based E-Commerce Sales Analytics system for resume and portfolio.
## Project Overview
This project simulates an E-Commerce sales database and performs analytical queries to extract business insights such as revenue, top-selling products, and customer behavior.

## Features
- Database Design
- Sales Analytics
- Customer Analysis
- Revenue Reports
- Business Insights
